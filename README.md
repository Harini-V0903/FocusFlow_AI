# FocusFlow_AI

##  Overview

FocusFlow AI is an AI-powered task planning agent that helps users organize their daily tasks intelligently. It classifies tasks into priority levels using a large language model (Groq LLaMA 3) and generates a structured, time-based daily schedule.

## Features

AI-based task prioritization (High / Medium / Low)

Smart daily schedule generation

Natural language task input

Powered by Groq LLM (LLaMA 3)

Simple and interactive Streamlit UI

## Tech Stack

Python

Streamlit

Groq API (LLaMA 3)

python-dotenv

## Learning Outcome

This project demonstrates:

AI agent design using LLMs

Prompt engineering for classification

Streamlit web app development

Real-world productivity system design
